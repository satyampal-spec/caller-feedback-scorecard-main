# 📞 Caller Feedback Scorecard

A simple, interactive tool to evaluate caller performance across 6 dimensions:
- Opening
- Storytelling  
- Probing
- Pricing Tackle
- Rebuttal Ranking
- Sentiment

## 🚀 How to Use

1. Open the tool
2. Enter caller name
3. Paste feedback in 3 sections
4. Click "Score This Call"
5. Get instant results with scores, issues, and pointers
6. Screenshot for WhatsApp

## 📊 What You Get

- Overall score (out of 10)
- Call sentiment (Excellent/Good/Average/Poor)
- What went wrong (auto-extracted issues)
- Key pointers (actionable improvements)
- Dimension scores (color-coded by performance)

Built for Even's inside sales team. Evaluate calls fast. Improve faster.
